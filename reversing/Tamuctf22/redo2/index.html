redo2

Description

Honestly, this is just a plain an simple ASM challenge. Best of luck.

Solution

Here, we are given an assembly file which is in intel format, so we first convert it into intel by adding a statement in the assembly file - ".intel_syntax noprefix" After this we compile the file - "gcc -m32 redo2.S -o redo2"

Observe the binary in binary ninja
