Solution :

  • Values of 'p','q','ciphertext','h'(encrypted hint which has the value of e) is given.
  • According to the code given, 'hint' is xored with 'n' to get the ciphertext 'h'.
  • To find the 'hint', 'h' is xored with 'n' to get 'hint', thus getting the value of 'e'. ```python= n = p*q hi = [i^n for i in h] l = [] for i in hi: x = chr(i) l.append(x) li = ''.join(l) print(li)
